# 🚨 AI Patient Vital Monitor

### 🏥 Project Purpose
- Built as a clinical-grade tool to detect "Code Blue" emergencies in real-time.
- Categorizes patient stability by analyzing seven critical physiological markers.

### 🧠 Core Features
- **AI Diagnostics**: Uses Scikit-Learn to predict "Stable" vs. "Critical" states.
- **Emergency UI**: Features pulsing "Siren" animations and a high-contrast ICU theme.
- **Smart Protocols**: Displays STAT medical actions like IV access and O₂ administration.

### 🛠️ Tech Stack
- **Frontend**: Streamlit with Custom CSS/HTML.
- **ML Backend**: Scikit-Learn, Pandas, and NumPy.
- **Storage**: Joblib for model and scaler serialization.

### 🚀 Quick Start
1. **Clone**: `git clone https://github.com/your-username/repo-name.git`
2. **Install**: `pip install -r requirements.txt`
3. **Launch**: `streamlit run app.py`
